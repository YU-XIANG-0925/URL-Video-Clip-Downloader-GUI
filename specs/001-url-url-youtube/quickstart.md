# 快速入門：多格式影片片段下載器

本指南將引導您完成使用此應用程式下載影片片段的步驟。

## 先決條件

- 已安裝 Python 3.11 或更高版本。
- 已安裝 `uv`。
- 已安裝 `FFmpeg` 並在系統的 PATH 中可用。

## 安裝

1.  **建立虛擬環境**

    ```bash
    uv venv
    ```

2.  **啟動虛擬環境**

    ```bash
    .venv\Scripts\activate
    ```

3.  **安裝依賴項**

    ```bash
    uv pip install yt-dlp
    ```

## 執行應用程式

```bash
python src/main.py
```

## 使用方法

1.  **啟動應用程式**：執行 `python src/main.py` 以開啟 GUI。

2.  **輸入影片 URL**：在 "URL" 輸入框中貼上您要下載的影片的 URL。

3.  **設定時間範圍**：
    - 在 "Start Time" 中輸入您想要的片段的開始時間 (格式為 HH:MM:SS)。如果留空，將從頭開始。
    - 在 "End Time" 中輸入您想要的片段的結束時間 (格式為 HH:MM:SS)。如果留空，將下載到影片結尾。

4.  **指定輸出位置**：
    - 點擊 "Browse" 按鈕選擇您要儲存檔案的資料夾。
    - 在 "Output Filename" 中輸入您想要的檔案名稱 (例如 `my-clip.mp4`)。

5.  **開始下載**：點擊 "開始轉換" 按鈕。

6.  **監控進度**：進度條和狀態文字將顯示下載和處理的進度。

7.  **完成**：任務完成後，您將在指定的輸出資料夾中找到您的影片片段。
